# 🚶‍♂️ Human Activity Recognition using Deep Learning

> 🧠 **A Deep Learning-based Human Activity Recognition (HAR) system that classifies physical activities using wearable sensor data.**

---

## 📌 Overview

Human Activity Recognition (HAR) is an important application of Artificial Intelligence that identifies human physical activities from wearable sensor signals. This project utilizes **Deep Learning** techniques to accurately classify various activities such as walking, sitting, standing, and climbing stairs using the **UCI HAR Dataset**.

---

## ✨ Features

- 📊 Data preprocessing and normalization
- 🛠️ Feature extraction and engineering
- 🧠 Deep Learning model for activity classification
- 📈 Model evaluation with multiple performance metrics
- 🎯 Confusion matrix visualization
- 🔍 Prediction on unseen sensor data
- ⚡ Scalable and modular project structure

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/human-activity-recognition.git
```

Move into the project directory:

```bash
cd human-activity-recognition
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the training script:

```bash
python src/train.py
```
