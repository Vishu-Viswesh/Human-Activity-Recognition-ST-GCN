# Human-Activity-Recognition-ST-GCN

## Overview

This project focuses on Human Activity Recognition (HAR) using sensor data collected from wearable devices. The objective is to classify different physical activities such as walking, sitting, standing, jogging, and climbing stairs using deep learning techniques.

---

## Features

- Data preprocessing and normalization
- Feature extraction
- Deep Learning model for activity classification
- Model evaluation using multiple metrics
- Confusion matrix visualization
- Prediction on unseen samples

---

## Dataset

Dataset Used:
- UCI Human Activity Recognition Dataset

Activities:
- Walking
- Walking Upstairs
- Walking Downstairs
- Sitting
- Standing
- Laying
